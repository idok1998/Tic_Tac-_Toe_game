# משחק איקס עיגול 
משחק איקס עיגול בפייתון לשני משתתפים. 

# איך לשחק הלוח מורכב מ-9 משבצות ממוספרות מ-1 עד 9:
'''
1 | 2 | 3
---+---+---
 4 | 5 | 6
---+---+---
 7 | 8 | 9
 '''
- כל שחקן בתורו בוחר את מספר המשבצת שבה הוא רוצה את למקם את הסימן שלו.
- - המטרה היא להשיג שלושה סימנים רצופים בשורה, בעמודה או באלכסון.
- - אם כל המשבצות מלאות ואין מנצח, המשחק מסתיים בתיקו.
